TexMindMapper

LaTeX file visualiser.

Features
  1. Parent-child relations as solid lines
  2. Reference relations as dashed lines
  3. Node size portrays relative word count
  4. Word count visible when hovering over a node
  5. Parses also projects consisting of multiple files
Processing your files

Simply select the files you want analysed. The script tries to add also the included texts, taken that the main file ends with "main.tex" (e.g. my_main.tex)
